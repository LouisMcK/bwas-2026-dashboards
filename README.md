# BWAS 2026 Dashboards

Static site for the BWAS 2026 World Cup prediction dashboards (served via GitHub Pages).

- `index.html` — landing page linking to each competition
- `original.html` / `ceej.html` — interactive dashboards (responsive; work on phone & desktop)
- `*-mobile.html` — extra-compact phone variants
- `*-race.html` — standalone leaderboard-race charts

These are self-contained HTML files (charts and data embedded). They're regenerated by
`build_bwas_dashboard.py` and assembled here by `publish_site.py` in the parent project.

Note: opening a dashboard, tapping a player name shows their card; the **Enter Scores**
tab recomputes everything live in your browser (changes are stored locally only).
